# Qr Menu

Or Menu is a system that provides convenience to cafes and customers.

Features:
- QR code generating
- Stock tracking
- Creating online menu
- Make an order, call waiter or check from a phone


Tech Stack: 
- Vue
- Tailwind
- Firebase
